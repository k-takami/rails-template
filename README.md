Rails template
==============

My Rails 4 application template

## Requirement
* Ruby ^2.2.0
* Rails ^4.2.0
* gibo<br> (for creating gitignore file)
```
$ brew install gibo
```

## Usage
```
rails new appname -BT -m https://raw.githubusercontent.com/kadoppe/rails-template/master/template.rb
```
